# FAQ

## 1. 项目放在哪个目录

```text
/root/project/mcp_query_table
```

## 2. 浏览器必须提前启动吗

不一定。

- 如果使用本地 CDP 地址，例如 `http://127.0.0.1:9222`，通常需要有已启动的浏览器
- 如果直接用 Playwright 拉起浏览器，可以不提前启动

## 3. 为什么查询结果为空或翻页失败

常见原因：

- 页面被切到移动端布局
- 网站要求登录但当前用户目录未登录
- 站点刚改版，选择器或接口已变化

## 4. 为什么无头模式下表现异常

无头模式更依赖 `user_data_dir`。如果没有显式指定，很多站点会要求重新登录。

## 5. 怎么确认服务启动成功

先本地运行：

```bash
cd /root/project/mcp_query_table
python3 -m mcp_query_table --format markdown --endpoint http://127.0.0.1:9222
```

如果是 SSE 模式，再检查：

```text
http://127.0.0.1:8000/sse
```

## 6. 为什么同一问题不同站点结果不同

因为不同站点的筛选口径、字段定义、分页实现和默认排序都可能不同。
